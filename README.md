# 👁️ Eye Catch - 아이트래킹 게임

눈으로 조작하는 혁신적인 모바일 웹 게임입니다!

## 🎮 게임 소개

Eye Catch는 스마트폰 카메라를 활용한 아이트래킹(시선 추적) 기술로 플레이하는 독특한 게임입니다. 화면에 나타나는 타겟을 눈으로 응시하여 점수를 획득하세요!

## ✨ 주요 기능

### 🎯 게임플레이
- **아이트래킹 기반**: 손을 사용하지 않고 눈으로만 플레이
- **직관적인 조작**: 타겟을 2초간 응시하면 캐치
- **난이도 증가**: 레벨이 올라갈수록 타겟이 작아지고 빠르게 생성
- **보너스 타겟**: 황금색 타겟은 2배의 점수!

### 📊 게임 모드
- **60초 타임어택**: 제한 시간 내에 최대한 많은 점수 획득
- **레벨 시스템**: 10초마다 레벨업으로 난이도 증가
- **점수 기록**: 최고 점수와 상위 10개 기록 저장

### 📱 모바일 최적화
- **PWA 지원**: 홈 화면에 설치 가능
- **반응형 디자인**: 모든 화면 크기에 최적화
- **오프라인 지원**: 설치 후 인터넷 없이도 플레이 가능
- **세로 모드**: 모바일 세로 화면에 최적화

## 🚀 시작하기

### 필요 조건
- 웹캠이 있는 디바이스 (스마트폰, 태블릿, PC)
- 최신 브라우저 (Chrome, Safari, Edge 등)
- 카메라 권한 허용

### 설치 및 실행

1. **로컬 서버 실행**
   ```bash
   cd /home/user/webapp
   python3 -m http.server 8000
   ```

2. **브라우저에서 접속**
   ```
   http://localhost:8000
   ```

3. **모바일에서 접속** (같은 네트워크)
   ```
   http://[PC의_IP주소]:8000
   ```

### PWA 설치 (선택사항)
1. 브라우저에서 게임 접속
2. 주소창의 "설치" 버튼 클릭
3. 홈 화면에 추가

## 🎲 게임 방법

### 1단계: 캘리브레이션
- 게임 시작 전 화면의 9개 점을 순서대로 클릭
- 눈의 위치를 보정하여 정확한 추적 가능

### 2단계: 게임 플레이
- 화면에 나타나는 원형 타겟을 눈으로 응시
- 2초간 응시하면 타겟 캐치!
- 진행 표시줄로 응시 시간 확인 가능

### 3단계: 점수 획득
- 일반 타겟: 10점
- 황금 보너스 타겟: 20점
- 60초 안에 최대한 많은 점수 획득

### 팁
- 고개를 움직이지 말고 눈만 움직이기
- 타겟 중앙을 정확히 응시하기
- 조명이 밝은 환경에서 플레이
- 카메라와 적절한 거리 유지

## 🛠️ 기술 스택

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던 스타일링 및 애니메이션
- **JavaScript (ES6+)**: 게임 로직 및 상태 관리

### 라이브러리
- **WebGazer.js**: 웹캠 기반 아이트래킹
- **Service Worker**: PWA 및 오프라인 지원
- **LocalStorage**: 점수 및 기록 저장

### 아키텍처
```
webapp/
├── index.html              # 메인 HTML
├── manifest.json          # PWA 매니페스트
├── sw.js                  # Service Worker
├── css/
│   └── style.css         # 스타일시트
├── js/
│   ├── app.js            # 앱 초기화
│   ├── game.js           # 게임 로직
│   ├── eyetracking.js    # 아이트래킹
│   └── storage.js        # 데이터 저장
└── assets/
    ├── images/           # 이미지 자산
    └── sounds/           # 사운드 자산
```

## 📊 게임 메커니즘

### 점수 시스템
- 일반 타겟: 10점
- 보너스 타겟: 20점 (10% 확률)
- 시간 제한: 60초

### 난이도 조절
- 레벨 1: 타겟 크기 60px, 3초마다 생성
- 레벨 2: 타겟 크기 55px, 2.8초마다 생성
- 레벨 3+: 계속 어려워짐
- 최소: 타겟 크기 30px, 1초마다 생성

### 타겟 수명
- 기본: 5초 동안 화면에 표시
- 레벨에 따라 감소 (최소 3초)
